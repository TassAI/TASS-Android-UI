# TASS Facenet Classifier

[![TASS Facenet Classifier](images/android.png)](https://github.com/TASS-AI/TASS-Android-UI/)

[![UPCOMING RELEASE](https://img.shields.io/badge/UPCOMING%20RELEASE-0.0.1-blue.svg)](https://github.com/TASS-AI/TASS-Facenet/tree/0.0.1)
